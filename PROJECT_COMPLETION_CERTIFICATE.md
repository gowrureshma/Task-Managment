GETTING STARTED
═══════════════════════════════════════════════════════════════════════════════

Quick Start (3 steps):

1. BACKEND
   cd backend
   npm install
   npm run seed
   npm run dev

2. FRONTEND (new terminal)
   cd frontend
   npm install
   npm start

3. BROWSER
   Open http://localhost:3000
   Login: admin@example.com / password123

Navigation:
  • Frontend:  http://localhost:3000
  • Backend:   http://localhost:5000
  • API Docs:  http://localhost:5000/api/health
